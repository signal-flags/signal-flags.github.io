Signal Flags Loader example

This example page shows how to use the Signal Flags Loader to include flags on any page without writing any JavaScript.

Getting started

Include the Signal Flags Loader script anywhere on the page (in the <head> is best if you use async as in the example below). 

<script async src="https://cdn.jsdelivr.net/npm/signal-flags@2/dist/signal-flags-loader.min.js"></script>

There are no other dependencies and we have not found any conflicts but note that the code is unlikely to work in any version of Internet Explorer other than IE 11.

Inline images

Insert a small image like this a in any text by writing <span class="signal-flags">[name]</span>, replacing [name] with the abbreviated name of the flag. Inline images are sized to fit their context so they are…

…bigger in a h n2 heading!

The list below shows the abbreviated flag names we use:

  • Letters A…Z a…z: a b c … d e f
  • Numerals 0…9 n0…n9: n0 n1 n2 … n7 n8 n9
  • Substitutes s1…s3: 1st s1 2nd s2 3rd s3
  • Racing flags black: black blue: blue green: green red: red yellow: yellow greenwhite: greenwhite blackwhite: blackwhite

Block elements

Use <div class="signal-flags">[name]</div> to display an image outside the flow of text. By default a block element will take up the whole width of the page so for smaller sizes you can either use a CSS layout framework like Bootstrap or set the width in CSS yourself. This page has the styles shown below in its <head>; note that pennants (numerals and the Answering Pennant) look good at 1.25× the width of rectangular and triangular flags. 

<style>
  div.signal-flags { width: 120px; }
  .signal-flags.pennant, .pennant .signal-flags { width: 150px; }
</style>
