Signal Flags Loader example

This example page shows how to use the Signal Flags Loader to include flags on any page without writing any JavaScript. There are no tricks here, although we do load the Bootstrap CSS to help lay things out.

Autoloading is achieved simply by including the Signal Flags Loader script anywhere on the page (in the head is best if you use async as in the example below). 

<script async src="https://cdn.jsdelivr.net/npm/signal-flags@2/dist/signal-flags-loader.min.js"></script>

Now you can insert a small image like this a in any text by writing <span class="signal-flags">[name]</span>, replacing [name] with the abbreviated name of the flag. Images are automagically sized to fit their context so they are…

…bigger in a h n2 heading!

Block elements

Use <div class="signal-flags">[name]</div> to display a larger image. By default a block element will take up the whole width of the containing element so you will normally need to set a width in somehow. This page has the styles shown below in its head; note that pennants (numerals and the Answering Pennant) look good at 1.25× the width of rectangular and triangular flags. 

<style>
  div.signal-flags { width: 120px; }
  .signal-flags.pennant, .pennant .signal-flags { width: 150px; }
</style>
